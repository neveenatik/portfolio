# Neveen Atik — Portfolio

Personal portfolio site built with Next.js, TypeScript and Tailwind CSS.

**Live site:** https://neveenatik.github.io/portfolio/

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Edit content (profile, skills, projects) in [`src/lib/data.ts`](src/lib/data.ts) — the site updates automatically.

## Deployment

The site is a static export deployed to **GitHub Pages** via GitHub Actions on every push to `main`.

```bash
npm run build   # static export to ./out
```

Live at https://neveenatik.github.io/portfolio/.
